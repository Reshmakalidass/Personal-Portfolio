# Personal-Portfolio
A fully responsive personal portfolio website developed using HTML, CSS, and JavaScript. This project highlights my professional profile, technical skills, and academic projects with smooth scrolling, animated UI components (AOS), and a clean dark-themed design. Optimized for both desktop and mobile devices and deployed using GitHub Pages.
# Personal Portfolio Website

This repository contains the source code for my personal portfolio website.  
The website is designed to showcase my professional profile, technical skills, and projects in a clean, modern, and responsive layout.

## Features
- Responsive design for mobile and desktop
- Smooth scrolling and animated sections using AOS
- Dark-themed modern UI
- Projects and skills showcase
- Contact section
- Deployed using GitHub Pages

## Technologies Used
- HTML5
- CSS3
- JavaScript
- AOS Animation Library
- Google Fonts
